# ReScript/React Project Template

## Installation

```sh
yarn
```

## Build

- Build: `yarn build`
- Clean: `yarn clean`
- Build & watch: `yarn dev`
- Deploy: `yarn deploy`

## Run

```sh
yarn start
```
